### Chap 05 : Filtering Operators

Using some filtering operators

- Ignoring operators
    - ignoreElements() :
    - elementAt(:)
    - filter(:)
- Skipping operators
    - skip(count)
    - skipWhile
    - skipUtil (waiting for a trigger)
- Taking operators
    - take(index)
    - takeWhile
    - takeUtil(waiting for a trigger)
- Distinct operators
    - distinctUntilChanged
    - distinctUntilChanged(_:) (distinct with condition)
